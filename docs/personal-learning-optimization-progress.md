# 个人学习版优化进度

## 已完成

- 聊天会话支持独立的运行参数面板，前端可保存模型、温度、上下文窗口、知识增强和系统提示词。
- 聊天请求已将 `sessionConfig` 一并传给 `agent-service`，不再只是前端本地存储。
- `agent-service` 会在运行时把会话配置注入模型调用，并按上下文窗口控制历史消息注入。
- 会话配置会写回后端会话元数据，切换会话后可以自动恢复。
- 聊天主视图已经接入会话运行参数面板。
- 模型网关新增“路由决策预览”，可查看候选模型、最终命中模型、降级情况和成本说明。
- 网关页新增“请求样本”切换和样本预览卡片，方便按场景模拟真实请求。
- Trace 阶段耗时已经改成真实埋点落库，监控页优先读取真实 `phase` 数据，不再默认依赖比例估算。
- 聊天消息会记录发送当次的运行参数快照。
- “继续生成 / 重新回答 / 分支会话”已继承原消息的运行参数，而不是误用当前面板配置。
- 聊天气泡展示消息级运行参数标签，便于回看某条回复当时使用的模型和参数。
- 聊天气泡新增单条消息收藏、导出、引用、继续追问等轻操作。
- 收藏逻辑已从简单布尔标记升级为结构化记录，能够保存消息内容、会话信息和参数快照。
- 新增“学习中心”页面，支持收藏列表、学习笔记、历史会话搜索三块能力。
- 学习中心支持把收藏转成笔记，也支持把会话搜索结果直接带入笔记编辑区。
- 学习中心已补齐标签分类、时间范围筛选、批量导出和学习资料包导出能力。
- 相似收藏会自动去重合并，并记录重复收藏次数，避免本地资料库越用越乱。
- 学习笔记已支持标签录入，收藏和搜索结果转笔记时会自动带入来源标签。
- 聊天页已新增“会话复盘”区块，可直接生成复盘模板、复制复盘并一键保存到学习笔记。
- 会话复盘会自动带入助手类型、模型策略和当前会话标签，形成聊天到学习沉淀的闭环。
- 学习中心已新增本地全文检索，可直接搜索收藏、笔记、标签和来源会话标题。
- 学习笔记已支持回链来源会话，复盘和搜索沉淀内容不再是孤立文本。
- 来源跳转已支持消息级定位，回到会话时会自动带上来源消息索引并高亮定位。
- 侧边栏已加入“学习中心”入口，路由已接入鉴权守卫。
- 前端相关测试已补齐，覆盖消息快照、继续生成继承、网关样本切换、学习中心渲染、标签筛选、批量导出、笔记保存、会话搜索和路由守卫。

## 当前剩余项

- 继续系统性清理前端历史遗留文案和异常提示，统一中文表达与状态文案。
- 为聊天页补更完整的学习闭环，例如消息摘要归档、会话复盘模板、基于收藏的二次提问入口。
- 后端如果后续继续演进，可考虑给学习中心增加服务端持久化接口，而不只依赖浏览器本地存储。
- 可继续补学习中心的服务端同步、跨设备资料漫游和全文索引，而不只依赖浏览器本地数据。
- 可继续补“复盘笔记二次编辑 / 复盘模板自定义 / 来源会话高亮定位”这类更细的个人化能力。
- 可继续补“来源消息高亮后的片段折叠 / 多来源比对 / 服务端同步定位信息”这类增强能力。
